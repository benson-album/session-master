# 天翼云盘 Cookie/存储分析报告

> 日期：2026-06-22  
> 工具：curl / whatweb / Kali Linux

## 一、HTTP 架构

| 端点 | 状态 | 服务器 | 说明 |
|:-----|:----:|:-------|:-----|
| `cloud.189.cn` | 301 | openresty | 跳转到 /web |
| `cloud.189.cn/web` | 301 | openresty | 跳转到 /web/ |
| `cloud.189.cn/web/` | 200 | openresty | 最终页面 |

> 重定向链路：cloud.189.cn → (301) → /web → (301) → /web/

## 二、安全头部

| 头部 | 值 |
|:-----|:----|
| `X-Frame-Options` | `SAMEORIGN` ⚠️（拼写错误，少I） |
| `X-XSS-Protection` | 缺失 ❌ |
| `Strict-Transport-Security` | 缺失 ❌ |
| `X-Content-Type-Options` | 缺失 ❌ |
| `Content-Security-Policy` | 缺失 ❌ |

> ⚠️ **重要警告：** `X-Frame-Options` 的值拼写为 `SAMEORIGN`（少字母 I），正确值应为 `SAMEORIGIN`。此拼写错误可能导致浏览器不识别该指令，从而无法有效防止点击劫持。

### 3.3 技术栈

| 组件 | 说明 |
|:-----|:------|
| Web 服务器 | openresty（基于 nginx + Lua） |
| TLS 证书 | 国产 WoTrus EV 证书（沃通） |
| SSO | 电信自有统一认证体系 |

> **WoTrus EV（扩展验证）证书**：国产 CA 机构沃通签发的 EV 证书，浏览器地址栏可显示企业名称（中国电信）。

## 六、结论

天翼云盘基于 openresty（nginx + Lua），使用国产 WoTrus EV TLS 证书。存在两次 301 重定向（→ /web → /web/）。**关键发现：** `X-Frame-Options` 值拼写错误 `SAMEORIGN`（少 I），可能导致浏览器不识别，存在点击劫持风险。未登录态无 Cookie 下发。认证体系需登录后确认。
