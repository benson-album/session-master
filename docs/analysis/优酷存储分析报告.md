# 优酷 Cookie/存储分析报告

> 日期：2026-06-22  
> 工具：curl / whatweb / Kali Linux

## 一、HTTP 架构

| 端点 | 状态 | 服务器 | 说明 |
|:-----|:----:|:-------|:-----|
| `www.youku.com` | 302 | Tengine/Aserver | 主站（跳转到 /ku/webhome） |
| `www.youku.com/ku/webhome` | 200 | Tengine/Aserver | 实际页面 |
| `login.youku.com` | — | — | 登录入口（无响应） |
| `passport.youku.com` | — | — | 已废弃 |

| 头部 | 值 |

## 三、认证机制

### 3.1 Cookie 体系

| Cookie | 域 | HttpOnly | 用途 |
|:-------|:---|:--------:|:-----|
| `_tb_token_` | `.youku.com` / `.taobao.com` | ✅ | 淘宝系令牌 |
| `_m_h5_tk` | `.youku.com` | ✅ | H5 会话密钥（核心认证） |
| `cookie2` | `.youku.com` | ❌ | 设备标识 |
| `lgc` | `.youku.com` | ❌ | 登录Cookie |
| `unb` | `.youku.com` | ❌ | 用户编号 |
| `ucn` | `.youku.com` | ❌ | 用户国家/地区 |
| `tfstk` | `.youku.com` | ❌ | 淘宝安全令牌 |
| `isI18n` | `.youku.com` | ❌ | 国际化标识 |
| `csrfToken` | `youku.com` | ❌ | CSRF 令牌（Session） |

### 3.2 localStorage

| Key | 用途 |
|:----|:------|
| `alitrackid` | 阿里跟踪ID |
| `mtop_jsonp_token` | Mtop API 令牌 |
| `mtop_sid` | Mtop 会话ID |
| `youku_config` | 播放配置 |
| `youku_userinfo` | 用户信息缓存 |

### 3.3 技术栈

- 后端：Tengine + Aserver（阿里自研）
- 全链路追踪：EagleEye（eagleeye-traceid）
- 登录：阿里统一登录（login.taobao.com 或 login.alibaba.com）
- API 网关：Mtop（阿里巴巴移动网关）

### 3.4 认证流程

```
访问 youku.com
  ↓
重定向到 /ku/webhome
  ↓
检查 _m_h5_tk + _tb_token_ Cookie
  ↓
存在 → mtop API 验证 → 返回登录态
不存在 → login.taobao.com → 扫码/账号登录 → 回调设置 Cookie
       ↓
阿里 SSO 跨域 Cookie（_m_h5_tk, _tb_token_, cookie2）
```

## 四、关联 Cookie 域

- `.youku.com` — 主域
- `.taobao.com` — 阿里SSO跨域
- `.alibaba.com` — 企业版SSO
- `.tmall.com` — 关联电商

## 五、已知 localStorage Key

```
alitrackid, mtop_jsonp_token, mtop_sid, youku_config, youku_userinfo
```

## 六、结论

优酷使用阿里统一登录体系，认证 Cookie 可能跨 `.youku.com` 和 `.taobao.com` 域。`_m_h5_tk` 为核心认证密钥，`_tb_token_` 为跨站令牌。localStorage 中的 `mtop_*` 系列为阿里网关令牌。
