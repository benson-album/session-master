### 3.3 技术栈

| 组件 | 说明 |
|:-----|:------|
| Web 服务器 | nginx 1.29.3 |
| API 网关 | Kong 3.1.1.3 enterprise-edition（Kong企业版） |
| TLS 证书 | 国产 cnTrus DV（中国专利 DV 证书） |
| SSO | 自有统一认证体系 |

> **Kong 企业版：** 使用 Kong API 网关的企业版本（3.1.1.3），具备商业级网关能力，包括流量控制、身份认证、日志等。
>
> **cnTrus DV 证书：** 国产 CA 机构签发的域名验证型证书。

### 3.4 认证流程（推测）

```
访问 www.123pan.com
  ↓
nginx → Kong API 网关 → 后端服务
  ↓
检查登录态（Cookie / Token）
  ↓
未登录 → 123云盘自有登录页面
       ↓
登录后设置认证 Cookie（域：.123pan.com）
```
