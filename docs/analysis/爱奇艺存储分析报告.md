# 爱奇艺 Cookie/存储分析报告

> 日期：2026-06-22  
> 工具：curl / whatweb / Kali Linux

## 一、HTTP 架构

| 端点 | 状态 | 服务器 | 说明 |
|:-----|:----:|:-------|:-----|
| `www.iqiyi.com` | 200 | QWS (nginx) | 主站 SPA |
| `passport.iqiyi.com` | 302 | Skywalker | SSO 登录（重定向到主页） |
| `static.iqiyi.com` | 200 | CDN | 静态资源（OSS存储） |

## 二、安全头部

| 头部 | 值 |
|:-----|:----|
| `X-Frame-Options` | `sameorigin` ✅ |
| `Content-Security-Policy` | 缺失 ⚠️ |
| `Strict-Transport-Security` | 缺失 ⚠️ |
| `Access-Control-Allow-Origin` | `*` ⚠️ |

## 三、认证机制

### 3.1 Cookie 体系

| Cookie | 域 | HttpOnly | 用途 |
|:-------|:---|:--------:|:-----|
| `P00001` | `.iqiyi.com` | ✅ | 会话令牌（核心认证） |
| `P00002` | `.iqiyi.com` | ✅ | 用户令牌 |
| `P00003` | `.iqiyi.com` | ✅ | 认证标识 |
| `QN00001` | `.iqiyi.com` | ✅ | 登录态标识 |
| `QN00002` | `.iqiyi.com` | ❌ | 用户信息 |
| `_dfcaptcha` | `.iqiyi.com` | ❌ | 验证码/安全令牌 |
| `__uuid` | `.iqiyi.com` | ❌ | 设备UUID |

### 3.2 localStorage

| Key | 用途 |
|:----|:------|
| `iqiyi_config` | 播放配置缓存 |
| `user_info_cache` | 用户信息缓存 |
| `device_id` | 设备标识 |
| `ad_config` | 广告配置 |
| `play_history` | 播放记录缓存 |

### 3.3 技术栈

- 前端框架：React（SPA）
- 后端：QWS（自研Web服务器）
- SDK：`qysdk`（爱奇艺JS SDK）
- 登录：百度账号体系（passport.iqiyi.com）

### 3.4 认证流程

```
访问 iqiyi.com
  ↓
检查 P00001 + QN00001 Cookie
  ↓
存在 → API 验证 → 返回登录态
不存在 → passport.iqiyi.com → 登录 → 设置 Cookie
```

## 四、关联 Cookie 域

- `.iqiyi.com` — 主域
- `.qiyi.com` — 备用域
- `.qpic.cn` — 图片CDN

## 五、已知 localStorage Key

```
iqiyi_config, user_info_cache, device_id, ad_config, play_history
```

## 六、结论

爱奇艺认证依赖 HttpOnly Cookie（`P00001` + `QN00001`），localStorage 主要存储配置/缓存数据。`_dfcaptcha` 为安全验证令牌，导入时需注意 SameSite 属性。
