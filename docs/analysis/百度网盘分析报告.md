# 百度网盘 Cookie/存储分析报告

> 日期：2026-06-22  
> 工具：curl / whatweb / Kali Linux

## 一、HTTP 架构

| 端点 | 状态 | 服务器 | 说明 |
|:-----|:----:|:-------|:-----|
| `pan.baidu.com` | 200 | nginx / BaiduCloud | 直接200无重定向 |

## 二、安全头部

| 头部 | 值 |
|:-----|:----|
| `X-Frame-Options` | 缺失 ❌ |
| `X-XSS-Protection` | 缺失 ❌ |
| `Strict-Transport-Security` | 缺失 ❌ |
| `X-Content-Type-Options` | `nosniff` ✅ |
| `Content-Security-Policy` | 缺失 ❌ |

## 三、认证机制

### 3.1 Cookie 体系

| Cookie | 域 | HttpOnly | 用途 |
|:-------|:---|:--------:|:-----|
| `BAIDUID` | `.baidu.com` | ❌ | 百度全局认证种子（核心） |
| `BAIDUID_BFESS` | `.baidu.com` | ❌ | BAIDUID 备用/兼容 |
| `PANPSC` | `.baidu.com` | ❌ | 网盘专属认证Token |
| `csrfToken` | `.baidu.com` | ❌ | CSRF 防护令牌 |
| `newlogin` | `.baidu.com` | ❌ | 新登录标识 |

- 全部 Cookie 跨子域 `.baidu.com`，百度系产品共享

### 3.2 localStorage

| Key | 用途 |
|:----|:------|
| `passLoginType` | 登录类型标记（已确认） |
| `bdstoken` | URL签名参数（API请求用） |
| `loginData` | 登录数据缓存 |
| `baidu_token` | 百度Token |

> 注：`bdstoken`、`loginData`、`baidu_token` 从 baidu.com 已有预设继承；`passLoginType` 为此站新增确认项。需登录态确认完整列表。

### 3.3 API 端点

| 端点 | 状态 | 说明 |
|:-----|:----:|:-----|
| `/api/list` | 200 | 需登录，返回文件列表 |
| `/api/check` | 404 | 不存在 |

### 3.4 签名机制

- **bdstoken**：URL 参数签名，用于 API 请求鉴权
- 所有敏感 API 需携带 bdstoken 参数

### 3.5 技术栈

- 后端：nginx + BaiduCloud（百度自研）
- SSO：百度统一登录（passport.baidu.com）
- 认证种子：BAIDUID 为全局认证标识

### 3.6 认证流程

```
访问 pan.baidu.com
  ↓
检查 BAIDUID Cookie
  ↓
存在 → 验证 BAIDUID + PANPSC → 返回登录态
不存在 → passport.baidu.com → 扫码/账号登录
       ↓
回调设置 BAIDUID、PANPSC 等 Cookie（.baidu.com 域）
```

## 四、关联 Cookie 域

- `.baidu.com` — 主域（百度系统一SSO跨域）

## 五、Preset 配置建议

```json
{
  "domain": "pan.baidu.com",
  "name": "BaiduPan",
  "nameZh": "百度网盘",
  "localStorageKeys": [
    "passLoginType",
    "bdstoken",
    "loginData",
    "baidu_token"
  ],
  "knownCookiePrefixes": [
    "BAIDUID",
    "PANPSC",
    "csrfToken",
    "newlogin"
  ],
  "cookieDomains": [".baidu.com"],
  "notes": "百度统一SSO认证。BAIDUID为全局认证种子，PANPSC为网盘专属。bdstoken为URL签名参数。需要登录态确认localStorage完整列表。"
}
```

## 六、结论

百度网盘使用百度统一SSO认证体系，BAIDUID 为全局认证种子，PANPSC 为网盘专属令牌。`bdstoken` 为 API 请求的 URL 签名参数。所有 Cookie 跨 `.baidu.com` 域。安全配置较弱，仅 X-Content-Type-Options 已开启。新增 `passLoginType` localStorage Key。
