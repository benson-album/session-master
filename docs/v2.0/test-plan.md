# SessionMaster v2.0 — 测试文档

> **版本**：v2.0-rc1
> **关联 PRD**：`docs/v2.0/PRD.md`
> **测试类型**：验收测试 + 回归测试
> **测试环境**：Chrome / Edge Manifest V3

---

## 1. 测试范围

### 1.1 覆盖范围

| 模块 | 测试重点 | 测试类型 |
|------|---------|:--------:|
| 锁定规则 | 三层锁定/解锁的正确性 | 功能测试 |
| 空白页横幅 | 增强文案是否按层级展示 | UI 测试 |
| btnP2pSaveConfig 校验 | 新增 deviceName 空值校验 | 功能测试 |
| 第二层解锁 | 同步连接操作在空白页下的行为 | 边界测试 |
| 第一层解锁 | 全局配置操作在空白页下的行为 | 边界测试 |
| 第三层锁定 | Cookie/保活填入在空白页下是否仍锁定 | 回归测试 |
| 正常页面 | 所有功能是否仍正常可用 | 回归测试 |

### 1.2 不覆盖的范围

- 后台功能逻辑（Cookie CRUD、WebRTC、加密、拦截等）— 与 v1.x 完全相同
- 服务器端功能 — 无变更
- 帮助文档 — 无变更

| # | 验收条件 | 验证方式 |
