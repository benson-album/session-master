# 决策记录

> **用途**：记录开发过程中所有影响范围、设计、质量的决策
> **原则**：如果没记录，就是没发生
> **位置**：`docs/v2.0/communication/coordination/`

---

## 决策分级

| 级别 | 代码 | 记录位置 | 谁发起 | 谁决策 |
|:----:|:----:|:---------|:-------|:-------|
| 🔵 微调 | `SCOPE_PATCH` | DE 的 impl-report 中记录 | DE | DE 自行决定 |
| 🟡 局部调整 | `SCOPE_ADJUST` | `coordination/E-编号.md` | PM | PM 开协调会 |
| 🟠 批次调整 | `SCOPE_CHANGE` | `coordination/F-编号.md` | PM→我 | 你决策 |
| 🔴 架构变更 | `SCOPE_REDESIGN` | `coordination/G-编号.md` | PM→我→PD→我 | 你决策 |
